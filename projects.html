What I've worked on during college

Senior Design

Portable Nutrient Data Analysis

 

 

My senior design project involves using a spectrometer to measure the wavelength of a sample using micro-discharge technology. Ultimately, this project will be able to detect nitrate samples in a liquid, and as such, will be very useful for portable water-quality testing.

 

Our team's job was to complete the signal path from the spectrometer to a mobile device. In order to accomplish this, we used a Raspberry Pi to collect data and send it to an Android device over Bluetooth. This data will then be stored in a database for online viewing through different clients.

MIPS Processor

A VHDL CPU

 

 

In CprE 389, we had to work together in groups of two in order to implement a functional and fully-featured MIPS processor in the VHDL hardware description language. When our project was completed, the processor could take an assembly program in binary and perform the Quicksort algorithm on a set of values in memory.

 

This project taught teamwork as well as how to break a project down into parts. In order to complete the project, it was necessary to come up with a design plan in order to finish the program on time. With the help of this roadmap, we were able to finish early and spend more time debugging and polishing the final product.

 

Beyond the team work aspect of this project, it also gave me an interesting view into how a computer processor works. While we didn't implement something as advanced as the entire x86 instruction set, it was still interesting to see how our design executed assembly code at an electrical level.

 

MIDIduino

MIDI Controlled ATmega328 Synthesizer

 

 

For this project, I worked on my own to create an audio synthesizer based on the ATmega328 processor. This specific chip is well known to electronic hobbyists as the CPU for the popular Arduino boards. Even though the processor is cheap and full of features, audio synthesis is not one of its strong suits. I was able to figure out relatively quickly how to play music through PWM (pulse width modulation), but the sound didn't have much character to it. Fortunately, I was able to find a project that used granular synthesis techniques to produce a greater range of sounds by mixing two PWM channels together.

 

Once I found this information, I decided to turn this project into a true instrument by adding MIDI capability. With this, I would be able to control the pitch of the synthesizer from both my computer as well as a electric keyboard. When all of the code was written, I designed a PCB (printed circuit board) from my schematics and soldered it all together.

 

This project was important to me because it was one of the first times that I was able to use my new-found knowledge from the university to build something I was interested in. Since I ended up designing a PCB instead of simply using a breadboard, I had to make sure that I got my design right the first time. These types of skills will be important at any job, since errors in designs cost both time and money to fix. Ultimately, this project helped hone both hard and soft skills and made me a better engineer.
